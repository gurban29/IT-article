# IT-article

# 1. Create a Virtual Environment
On Windows:
python -m venv venv

On macOS/Linux:
python3 -m venv venv

# 2. Activate the Virtual Environment
On Windows:
.\venv\Scripts\activate

On macOS/Linux:
source venv/bin/activate

# 3. Install Django
pip install django

or install dependencies from the requirements.txt file:
pip install -r requirements.txt
